# 快捷键说明：
## Ctrl + Z ： 撤销上一个操作；
### 完成绘制的多边形被视为一个整体进行撤销.
### 没有可撤销对象时出现 beep 的警告提示音.
### 没有实现 redo 功能.
        
# 鼠标按键说明：
## 鼠标左键单击：添加点.
## 鼠标左键双击：结束绘制多边形.
### 只有当绘制的图形是多边形时才可以结束.
### 如果只有一个点或者线是无法通过双击结束绘制.
## 鼠标右键单击：显示右键菜单.
### Finish Part:   结束当前轮廓的绘制.
### Finish Sketch: 结束当前多边形的绘制.
        
# 附加功能说明：
## 实现了橡皮筋功能.
### 同时刷新当前多边形形状，与 ArcGIS 一致.
## 支持 MultiPolygon 类型对象的创建.
## 实现了点的引力场.
### 因为不支持放大缩小，所以引力场比较小，只有四个像素大小.
        
# BUG：
